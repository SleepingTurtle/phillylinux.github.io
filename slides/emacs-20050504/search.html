Searching and Replacing

C-s Incremental search forward
C-r Incremental search backward
C-s RET Non-incremental search forward
C-r RET Non-incremental search backward
M-C-s Incremental search forward by regexp
M-C-r Incremental search backward by regexp
M-C-s RET Non-incremental search forward by regexp
M-C-r RET Non-incremental search backward by regexp
M-x replace-string Unconditionally replace a string with another
M-x replace-regexp Unconditionally replace a pattern with a substitution
M-% Query replace
M-C-% Query replace regexp
M-x occur Find all occurrences of a pattern - powerful
