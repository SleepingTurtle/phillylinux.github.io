Outline Minor Mode (cont'd)

Now that we've solved the interface problem, what can we do with it?

M-o t Hide all body lines (show only headings)
M-o e Expand one section
M-o c Collapse one section
M-o o Expand this one section, hide all others
M-o n Move to next heading
M-o p Move to previous heading
