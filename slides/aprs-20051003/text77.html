Relay,Wide vs Smart paths

  • Relay, Wide will be repeated by any Relay and Wide digipeaters, without checking to see if this packet has already been transmitted.

  • WIDEn-N will be intelligently repeated by WIDEn-N digipeaters

    • Retain checksum of packet
    • Don't retransmit packet with same checksum over next 28 seconds
    • Can include firewall for large N (Wide4-4 + )
